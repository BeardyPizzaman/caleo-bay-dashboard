# Chancellor Memory Care Operations Academy (Website)

## Open Locally
From this folder, run:

```bash
python3 -m http.server 8080
```

Then open:

- http://127.0.0.1:8080/index.html

## What Is Included
- 10-module curriculum blending industry fundamentals with Chancellor-specific operating playbooks
- Interactive module roadmap with progress tracking (localStorage)
- Industry Foundations Lab (care continuum, economics, payer flow, regulatory exposure, growth/risk)
- Sacramento + Oregon location comparison lab
- Revenue/payment scenario simulator by community
- Final 14-question graded knowledge check (industry + Chancellor operations)
- Source reference links for policy updates
