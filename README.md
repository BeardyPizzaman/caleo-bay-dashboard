# Caleo Bay Investment Analysis Dashboard

A comprehensive, interactive financial analysis dashboard for evaluating senior living facility investments.

## üöÄ Features

### Financial Analysis
- **Real-time Valuation Calculator** - Dynamic cap rate sensitivity analysis
- **Debt Modeling** - LTV scenarios, DSCR calculations, and levered returns
- **5-Year Pro Forma** - Forward-looking projections with adjustable assumptions
- **Expense Breakdown** - Detailed operating expense analysis
- **T-12 Performance** - Historical revenue and occupancy trends

### Investment Tools
- **IRR Calculator** - Internal Rate of Return for multi-year holds
- **Cash-on-Cash Return** - Levered yield calculations
- **Scenario Comparison** - Save and compare multiple investment scenarios
- **Price Per Unit Metrics** - Unit-level economics analysis

### User Experience
- **Dark Mode** - Full dark theme support with persistent preferences
- **Keyboard Shortcuts** - Navigate efficiently with hotkeys
- **Export Functionality** - Download analysis data as JSON
- **Print-Friendly Layout** - Optimized for PDF generation
- **Responsive Design** - Works on desktop, tablet, and mobile
